Legal Partners – Website Redesign

A clean and conversion-focused UI/UX redesign for a lawyer consulting website.

📌 Overview

This project redesigns a local law firm website to improve:

Clarity of services
User experience
Lead generation

The goal is to turn visitors into potential clients.

🎯 Objective
Clearly present legal services
Build user trust
Encourage users to take action (book / contact)
Ensure mobile-friendly design
👥 Target Users
Individuals needing legal help
Small business owners
Clients looking for quick consultation
📄 Pages
🏠 Homepage (services + CTA)
📑 Service Page (details + FAQ)
📞 Contact Page (form + details)
✨ Key Features
Strong call-to-action (CTA)
Simple and clean layout
Testimonials for trust
Easy contact form
Responsive design
🧠 UX Decisions
CTA placed at the top for quick action
Minimal form fields to reduce friction
Clear layout for easy navigation
Trust elements to build confidence
🛠️ Tools Used
Figma
Google Fonts
Coolors
